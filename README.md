# Aplicación de Calendario de Entrenamiento

Esta aplicación web sencilla permite marcar los días del mes en los que realizas entrenamiento. Al hacer clic sobre un día del calendario, se tacha para indicar que entrenaste ese día. Si vuelves a hacer clic se destacha.

## Uso
1. Abre el archivo `app/index.html` en tu navegador web.
2. Haz clic sobre los días correspondientes a tus sesiones de entrenamiento.

El calendario se genera para el mes actual y no guarda la información al cerrar la página.
